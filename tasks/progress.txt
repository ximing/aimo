## Codebase Patterns
- Use @rabjs/react for frontend state management with observable stores
- Service classes extend Service from @rabjs/react and use bindServices() for React integration
- API calls use the request wrapper with typed responses
- SM-2 standard formula: EF' = max(1.3, EF + (0.1 - (5-q)*(0.08 + (5-q)*0.02)))
  - q=5: +0.1, q=4: 0 (unchanged), q=3: -0.14, q=1: -0.54 (often clamped to 1.3)
- Interval must use OLD easeFactor (before update) to avoid double penalty on fuzzy ratings
