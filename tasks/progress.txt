## Codebase Patterns
- Use Drizzle ORM for MySQL schema - all schemas in schema/ directory must be exported from index.ts
- Always build server (`pnpm build`) before generating migrations (`pnpm migrate:generate`)
- Migration files are generated in drizzle/ folder at server root

- **Learnings for future iterations:**
  - The VARCHAR(191) length limit is due to MySQL utf8mb4 index size limit (767 bytes / 4 bytes per char = 191)
  - Drizzle migration adds nullable columns without DEFAULT NULL - existing rows will have NULL values
  - Build must happen before migration generation since drizzle-kit reads from dist/ folder
